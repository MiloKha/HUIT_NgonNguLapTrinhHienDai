2. Về website

Chức năng chính:

Trang chủ: Hiển thị các sản phẩm được giới thiệu. Khách hàng có thể tìm kiếm sản phẩm và truy cập vào các trang chi tiết.

Trang chi tiết sản phẩm: Hiển thị đầy đủ thông tin về từng sản phẩm như hình ảnh, mô tả, giá cả, size, màu sắc, v.v. Khách hàng có thể thêm sản phẩm vào giỏ hàng từ trang này.

Giỏ hàng và thanh toán: Khách hàng có thể quản lý các sản phẩm trong giỏ hàng, cập nhật số lượng, xem tổng giá trị đơn hàng và tiến hành thanh toán an toàn.

Quản lý sản phẩm: Trang dành cho quản trị viên, cho phép thêm/sửa/xóa thông tin các sản phẩm, quản lý danh mục sản phẩm.

Quản lý đơn hàng: Trang quản lý các đơn hàng của khách hàng, cho phép xem trạng thái đơn hàng, gửi email xác nhận đơn hàng, v.v.

Về mặt kỹ thuật:

Django: Sử dụng Django framework để xây dựng ứng dụng web, quản lý routing, views, models, templates.

Visual Code: Sử dụng trình soạn thảo Visual Studio Code để phát triển code, debug, quản lý source code.

SQLite: Sử dụng SQLite database để lưu trữ thông tin sản phẩm, đơn hàng, khách hàng.

Python: Sử dụng ngôn ngữ lập trình Python để lập trình backend, xử lý logic nghiệp vụ.
